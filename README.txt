Setup and manage your whos.amung.us real-time / live stats very easily.

== Description ==

This is a plugin that will allow you to setup and manage your whos.amung.us stats very easily:
* With just a few clicks you can setup the plugin and it's widget and test the various displaying options to find which suits your website better!
* View stats inside your wordpress instaltation - You will be able to view live stats about your website insider your wordpress instalation

This plugin was developed by [Miguel Pinho](http://MiguelPinho.com/ "").
Visit [Whos.Amung.us](http://whos.amung.us/ "Real-time stats for your website or blog.") and get Real-time stats for your website or blog!


== Installation ==

Here is a simple step-by-step of the instalaction and configuration process:

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the widgets section and setup the widget
4. Go to the "Whos Amung Us" tab in the admin panel and view your stats
